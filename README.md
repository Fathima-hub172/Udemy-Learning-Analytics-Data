# Udemy Courses Analysis 📊

## Overview
This project explores Udemy course data to uncover insights into course popularity, pricing, ratings, and student engagement.  
It combines **Python (data cleaning & exploration)** and **Power BI (interactive dashboards)** to demonstrate advanced analytics and visualization skills.

---

## Features
- **Python Analysis**
  - Data cleaning with Pandas & NumPy
  - Exploratory Data Analysis (EDA) with Seaborn, Matplotlib, Plotly
  - Statistical insights: mean, std, correlations
- **Power BI Dashboard**
  - KPIs: Total Courses, Average Rating, Total Students
  - Visuals: Category distribution, Price vs Enrollment, Ratings trend
  - Interactive filters: Category, Level, Year
- **Business Insights**
  - Which categories attract the most learners
  - Impact of pricing and discounts on enrollments
  - Trends in ratings across subjects

---

## Tech Stack
- **Python**: NumPy, Pandas, Matplotlib, Seaborn, Plotly  
- **Power BI**: interactive visuals  
- **Dataset**: Udemy course dataset (CSV/Excel)
